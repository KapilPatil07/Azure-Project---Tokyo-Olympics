
Tokyo Olympics :
===============


Cloud: 
======
Azure Cloud



Data Source :
============
Github / Kaggle



Services Used :
==============

The list of Azure services used to build this project are as follow -

1. Azure Storage Account: Azure DataLake Gen 2
2. Azure Data Factory
3. Azure Databricks
4. Azure Synapse Analytics
5. Azure Key Vault



Summary :
=========

- In this project, we have developed an end-to-end Azure Data Engineering Pipeline
  using the data set named Tokyo Olympics which is present in our data source named
  Github or Kaggle.


- So in this project, we have extracted the data from data source i.e Github/Kaggle 
  in our case.

- Using ADF pipeline we loaded the raw data into adls gen2
